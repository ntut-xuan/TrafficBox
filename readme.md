# TrafficBox

TrafficBox 是一款能夠利用當前 GPS 坐標資訊，取得周遭交通資訊整合的 Android APP，適合判斷要乘坐哪種交通，以及在人生地不熟的情況下能夠取得周遭的交通資訊。



## 作者

國立臺北科技大學 資訊工程系 [黃漢軒](https://ntut-xuan.github.io)
